It Publishes Itself

This isn't a copy generator that dumps text into a Google Doc. Parts of this system deploy directly to production.

Help articles publish directly to the Bubble CMS via API -- written, formatted, and live without anyone logging in.
In-app notifications push live to users automatically, targeting the right segments with the right message.
Marketing images are generated, hosted on imgbb, and embedded in the content. No manual upload step.
The feature tracker updates automatically across every channel, so nothing falls through the cracks.

The system even scans screenshots for real user names and replaces them with fictional ones before generating marketing images. Privacy-safe by default, no manual review needed.

How It Works

Step 01
Intake
Describe the feature in plain language. The system asks clarifying questions, checks for screenshots, and identifies which user segments care about this update.
Step 02
Core Messaging
Creates one headline, one subheadline, and one description. This is the single source of truth for the feature's message. You approve it before anything else gets written.
Step 03
Channel Routing
Evaluates every channel against the feature's characteristics. Marks each one as active or skip, with a reason. Not every feature needs all 15 channels, and the system knows the difference.
Step 04
Content Generation
For each active channel, reads the channel-specific format rules, brand voice guide, and image specs. Writes the copy and generates images. Each piece sounds like it was written for that platform, because it was.
Step 05
Publishing
Auto-deploys where possible -- Bubble CMS articles, in-app notifications, hosted images. Everything else is formatted and ready to paste into the channel. One review, then it goes out.

The Pattern

Every business ships updates. Most announce them inconsistently -- different tone on social media, missing channels, or just skipping the announcement entirely because writing it for 15 places takes too long. The updates that do go out feel scattered. The ones that don't go out are missed opportunities.

This system makes sure every feature gets full, consistent coverage across every channel, on-brand, every time. It doesn't replace judgment -- you still approve the messaging and review the output. It replaces the repetitive, time-consuming work of adapting the same message for a dozen different formats.
